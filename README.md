## vue-notes-list
SPA-приложение по добавлению и сохранению заметок.  
В приложении есть страницы авторизации, регистрации, список заметок, добавление заметки.

**Frontend:** Vue/Vuex/Vue-Router + materializecss;  
**Backend:** Firebase.
